eureka环境搭建与目录

环境搭建

  1. github下载eureka，切到v1.7.2tag

  2. 搭建gradle环境，v1.7.2版本使用2.1版本

  3. win运行gradlew.bat下载依赖

  4. 导入IDE

    eureka调试环境

  5. 启动eureka-server测试代码

在eureka-server找到测试代码

  1. 启动eureka-client测试代码

eureka目录

  • eureka-client
    eureka的客户端，注册到eureka上面去的一个服务，就是一个eureka client，无论是你要注册，还是要发现别的服务，无论是服务提供者还是服务消费者，都是一个eureka客户端。
  • eureka-core
    eureka的服务端，其实就是eureka的注册中心
  • eureka-resources
    基于jsp开发的eureka控制台，web页面，上面你可以看到各种注册服务
  • eureka-server
    eureka-client、eureka-core、eureka-resources打包成了一个war包，也就是说eureka-server自己本身也是一个eureka-client，同时也是注册中心，同时也提供eureka控制台。真正的使用的注册中心
  • eureka-examples
    eureka使用的例子
  • eureka-test-utils
    eureka的单元测试工具类

web.xml解读

  1. listener : com.netflix.eureka.eurekaBootStrap
    listener一般用于初始化
  2. filter : statusFilter
    负责状态相关的处理逻辑
  3. filter : requestAuthFilter
    对请求进行授权认证的处理的
  4. filter : rateLimitingFilter
    负责限流相关的逻辑的
  5. filter : gzipEncodingEnforcingFilter
    gzip，压缩相关的；encoding，编码相关的
  6. filter : jersey
    jersey框架的一个ServletContainer的一个filter

statusFilter和requestAuthFilter为通用处理逻辑，对所有请求开放
rateLimitingFilter默认关闭，按需要自行开启
GzipEncodingEnforcingFilter，对/v2/apps相关请求生效
